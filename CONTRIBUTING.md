# Contributing

Thanks for contributing! This scaffold is intentionally small. If you plan to contribute:

- Follow Unity best practices and keep editor-specific binary files out of commits when possible (use .gitignore).
- Do not commit third-party proprietary binaries (Agora SDK) — instead add them to Assets/Plugins locally and mention the version in PR description.
- Open issues for feature requests (avatar assets, mobile optimizations, Agora integration).
